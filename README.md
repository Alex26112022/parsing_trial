## Перед началом работы проверьте, что у Вас на компьютере установлена последняя версия Google-Chrome!  

### Запустите файл main.py в корне проекта!
### После успешного поиска данные запишутся в файл sud_dela.csv (новые данные добавляются в конец файла)
* ключевые слова поиска определяются переменной search_text в main.py
### Логи проекта формируются в папке logger(можно настроить):
* максимальный объем файла определяется переменной rotation в main.py (по умолчанию 1 MB)
* при превышении указанного объема файл сжимается в zip-архив и создается новый лог-файл с меткой времени.
* максимальное время хранения лог файлов определяется переменной retention в main.py (по умолчанию 365 дней).
